# Python Web Scraper

This is a simple Python web scraping project.

## Features
- Takes a website URL as input
- Scrapes all H2 headings
- Saves data into SQLite database

## Technologies Used
- Python
- BeautifulSoup
- Requests
- SQLite

## How to Run

1. Install dependencies
pip install requests beautifulsoup4

2. Run the script
python scraper.py

3. Enter a website URL
